CSS3 Basic

2019年7月13日
| Front-end
主要针对CSS3 本篇是笔记，因此覆盖会不全面 ####Reference w3school border div { border:2px solid #ffffff; border-radius:25px; -moz-border-radius:25px; /* Old Firefox */ } border控制边界的大小，形状和颜色 border-radius控制着边框的弯曲的半径，随着数字越大，边框会越弯 transition div { width:100px; height:100px; background:yellow; transition:width 10s; -moz-transition:width 2s; /* Firefox 4 */ -webkit-transition:width 10s; /* Safari and Chrome */ -o-transition:width 2s; /* Opera */ } div:hover { width:300px; } 这个的效果就是在div上悬停，就会将div的宽度在一定时间内切换到另一个指定的宽度 可以看到，需要在transition那里声明哪个属性变化 如果要有多个属性变化(比如 width和height都变) div { width:100px; height:100px; background:yellow; transition:width 2s, height 2s; -moz-transition:width 2s, height 2s, -moz-transform 2s; /* Firefox 4 */ -webkit-transition:width 2s, height 2s, -webkit-transform 2s; /* Safari and Chrome */ -o-transition:width 2s, height 2s, -o-transform 2s; /* Opera */ } div:hover { width:200px; height:200px; transform:rotate(180deg); -moz-transform:rotate(180deg); /* Firefox 4 */ -webkit-transform:rotate(180deg); /* Safari and Chrome */ -o-transform:rotate(180deg); /* Opera */ } 可以看到只是简单的用逗号隔开就可以了……
