SQL 进阶

2019年4月23日
| 数据库
| 阅读

只针对Mysql的SQL语句

##创建TABLE

CREATE TABLE user
(
    id int PRIMARY KEY,
    username varchar(255) NOT NULL UNIQUE,
    password varchar(255) NOT NULL
);

约束也可以这样写

CREATE TABLE user
(
    id int,
    username varchar(255) NOT NULL,
    password varchar(255) NOT NULL
    City varchar(255) DEFAULT 'Sandnes'
    UNIQUE(username)
    PRIMARY KEY(id)
    FOREIGN KEY (P_id) REFERENCES Persons(P_id)
    
)

约束

  • NOT NULL

  • UNIQUE

  • PRIMARY KEY(NOT NULL和UNIQUE的结合)

  • FOREIGN KEY:保证一个表中的数据匹配另一个表中的值的参照完整性

  • CHECK

  • DEFAULT 默认值，在没有赋值的情况下就是默认值

NOT NULL

强制不接受NULL，如果不向该字段添加值，就无法插入新纪录或者更新记录

UNIQUE

保证某个列不会出现相同的值，比如说用户名要取不同的，所以在数据库中就要加上UNIQUE强制不同

每个表中可以有多个UNIQUE，但是PRIMARY KEY只能有一个

表已经创建后，添加删除约束

ALTER TABLE tb_name ADD UNIQUE(id)

ALTER TABLE tb_name DROP INDEX column_name

PRIMARY KEY

操作和UNIQUE一样，只是在撤销约束的时候有点不同

ALTER TABLE tb_name DROP PRIMARY KEY;

FOREIGN KEY

ADD

ALTER TABLE tb_name
ADD FOREIGN KEY (column_name)
REFERENCES tb1_name(column1_name)

DROP

ALTER TABLE tb_name
DROP FOREIGN KEY

DEFAULT

ADD

ALTER TABLE Persons
ALTER City SET DEFAULT 'SANDNES'

DROP

ALTER TABLE Persons
ALTER City DROP DEFAULT
