题目

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

示例 1:

输入: ["flower","flow","flight"]
输出: "fl"

示例 2:

输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。

说明:

所有输入只包含小写字母 a-z 。

我的解答

func longestCommonPrefix(strs []string) string {
	if len(strs) == 0 {
		return ""
	}
	if len(strs) == 1 {
		return strs[0]
	}

	shortestLen := len(strs[0])

	for _, str := range strs {
		if len(str) < shortestLen {
			shortestLen = len(str)
		}
	}

	var finalStr string

	for i := 0; i < shortestLen; i++ {
		tmp := strs[0][i]
		for j := 1; j < len(strs); j++ {
			if tmp != strs[j][i] {
				return finalStr
			}
		}
		finalStr += string(tmp)

	}
	return finalStr

}

使用垂直检测，一个个字符的检测，时间复杂度是O(n*m) n是最短字符串的字符个数,m是字符串的个数

最优解答

func longestCommonPrefix(strs []string) string {
	size := len(strs)
	if size == 0 {
		return ""
	}

	minPos := 0
	minLen := len(strs[0])
	for i := 0; i < size; i++ {
		if minLen > len(strs[i]) {
			minLen = len(strs[i])
			minPos = i
		}
	}

	for j := 0; j < minLen; j++ {
		temp := strs[minPos][j]
		for i := 0; i < size; i++ {
			if i == minPos {
				continue
			}
			if temp != strs[i][j] {
				if j == 0 {
					return ""
				}
				return strs[i][:j]
			}
		}
	}
	return strs[minPos]

}

难疑点&核心

这里有很多其它的方法，包括字典树(相对复杂些，但是字典树在路由分配上也有用到，是值得学习的)、水平扫描等

垂直扫描是比较简单的一种
