关于select case

    runtime.GOMAXPROCS(1)
    int_chan := make(chan int, 1)
    string_chan := make(chan string, 1)

    int_chan <- 1
    string_chan <- "hello"
    select {
    case value := <-string_chan:
        fmt.Println(value)
    case value := <-int_chan:
        panic(value)
    }

上面代码两个case都满足，会伪随机的执行一个

关于defer

func DeferCalc() {
	a := 1
	b := 2
	defer calc("1", a, calc("10", a, b))
	a = 0
	defer calc("2", a, calc("20", a, b))
	b = 1

	time.Sleep(1 * time.Second)
}

func calc(index string, a, b int) int {
	ret := a + b
	fmt.Println(index, a, b, ret)
	return ret
}

输出会是

10 1 2 3
20 0 2 2
2 0 2 2
1 1 3 4

首先注意第一个点，参数的传递是值传递

    a := 1
	b := 2
	defer calc("1", a, calc("10", a, b))

这里传递进去的a,b就是当时的a,b变量的值，和以后a,b的值无关

第二个点就是defer 函数里面嵌套的函数会先执行，不会等到defer开始了才执行

第三个点就是defer的执行顺序是栈的形式
