今天来学习一下布局方式

现在主流的布局方式flexbox

Reference

https://css-tricks.com/snippets/css/a-guide-to-flexbox/

Basic

display: flex就将这个block变成了flexbox

首先，我们要先搞明白container和item的关系

一般来说item是最小的单位，也就是一个东西(比如：按键…)

container就是我们的block，这个就是在页面设计的时候考虑要如何放置我们的block，container是可以嵌套的

Axis

For rows, the main axis is a horizontal line and for columns it is a vertical line.

首先要搞懂轴的概念

轴的方向是和flex-direction相关联的

根据参数我们可以很快确定轴的方向

row             //main 左到右 cross 上到下
column          //main 上到下 cross 左到右
row-reverse     //...
column-reverse  //...

比如flex-direction: row，那么就按照上面这个图的方向

  • 主方向为横向(也就是item的排列方向)

  • cross方向为纵向

我们后面说到的属性可能会和轴有联系

而这篇文章也是假设了flex-direction: row

flex-wrap

可选参数

nowrap (default)    //这个就是将所有item按一个方向排列并且不会，并且只有一行
wrap                //这个相对上面就会进行换行
wrap-reverse        //反转,就是和wrap反过来，从底部到顶部

可以看出这个和两个轴都有关系

先在main轴排列，不过空间可以wrap到cross轴

flex-flow

这是一个集合

flex-flow: <flex-direction> || <flex-wrap>

默认值是：row nowrap

justify-content | align-items |c align-content

刚学习时会常常把这两个搞混，所以一起讲，作对比

并且这两个布局的参数也基本一致

首先justify-content是以main轴作用的

可以看到这个参数影响的效果可以等同于这几个物体在左右移动(因为是row)

所以反过来，align-items就是在cross轴移动

说移动只是为了好理解，实际上也并不是简单的移动可以解释清除的

所以只要记住这两个布局和轴之间的关系就很好区分了

还有一个align-content

flex-start
flex-end
center
space-between
space-around
stretch

这个就是排列cross line的

order

排列item在container的顺序，越小越前面

flex-grow

定义一个item的增长比例

这个参数会自动填充空余空间

比如大家都设置为1，那么就是平均分了

align-self

作用效果和align-items是差不多的，差别在于align-items的作用是整个container的items，而这个的话是对一个item的，所以这个能够overwrite容器给的align
