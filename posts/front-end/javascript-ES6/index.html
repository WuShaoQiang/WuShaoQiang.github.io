let vs var

在ES6之前还没有let

var是有一些很严重的缺点的，比如

可以重复定义同名的变量而不会抛错

经常想定义一个局部变量，变成了一个全局变量，也就是shadow和我们平时熟悉的方式不一样

所以我们在定义变量的时候就用let，因为let就是修复了var的一些缺点，和平时的用法一样

const

这个和let差不多，只是只能赋值一次(定义的时候就要赋值)

并且建议const用全部大写的格式

开发建议是：如果变量是不会重新赋值的，那么最好是使用const，除非是预测到要重新赋值就用let

在这里还要强调的一点，赋值是值这个变量的赋值，如

const s = [5,6,7]
s = [1,2] //出错，重新赋值
s[2] = 10 //可行，这个不是对s的重新赋值

同理的还有Object、function

可以防止一个Object的属性被修改

Object.freeze(object_name);

filter map reduce

A function that accepts up to three arguments. 
The filter method calls the callbackfn function one time for each element in the array.

示例

const realNumberArray = [4, 5.6, -9.8, 3.14, 42, 6, 8.34, -2];
const squareList = (arr) => {
  "use strict";
  // change code below this line
  const squaredIntegers = arr.filter((num) => num > 0 && num % parseInt(num) === 0).map((num) => Math.pow(num,2))
  // change code above this line
  return squaredIntegers;
};
// test your code
const squaredIntegers = squareList(realNumberArray);
console.log(squaredIntegers); //16,1764,36

对arr使用filter过滤出正整数，然后用map整合数据

还有就是箭头函数返回一个Object可以这样写

(name,age) => ({name,age})

Function params

设置默认参数很简单

fcuntion getName (name = "yourname"){
    ...
}

当没有参数传入的时候就会调用默认参数

设置剩余参数

const sum = (function() {
  "use strict";
  return function sum(...args) {
    return args.reduce((a, b) => a + b, 0);
  };
})();
console.log(sum(1, 2, 3)); // 6

传入的剩余参数以数组的形式显示

...只在function的参数或者数组的遍历上起作用

Math.max(...arr)

Destructuring from object

以前都是用.来获取Object的元素的

现在可以用一个解构的方法

var voxel = {x: 3.6, y: 7.4, z: 6.54 };
const { x, y, z } = voxel; // x = 3.6, y = 7.4, z = 6.54

或者重命名
const { x : a, y : b, z : c } = voxel // a = 3.6, b = 7.4, c = 6.54

同理，如果是nest后的结构体

const a = {
  start: { x: 5, y: 6},
  end: { x: 6, y: -9 }
};
const { start : { x: startX, y: startY }} = a;
console.log(startX, startY); // 5, 6

当我们想要给一个函数传入一个Object，但我们可能只用到其中的一两个字段，那么可以这样写参数

// 假设stats有max和min字段，并且还有别的字段，那么有两种方法可以获得max和min字段

// 第一种 直接传入Object
function half(stats) {
    // use function argument destructuring
    return (stats.max + stats.min) / 2.0;
};

// 第二种直接指定字段，外面调用的时候传入Object
({max,min}) =>{return (max+min)/2}

Destructuring from array

const [a, b] = [1, 2, 3, 4, 5, 6];
console.log(a, b); // 1, 2

const [a, b,,, c] = [1, 2, 3, 4, 5, 6];
console.log(a, b, c); // 1, 2, 5

在这里不能够指定某一个元素给变量赋值

所以如果我们要交换两个数

(() => {
    [a,b] = [b,a]
})();

//这是一个函数调用的方法

还有一种用...进行赋值的

const [a, b, ...arr] = [1, 2, 3, 4, 5, 7];
console.log(a, b); // 1, 2
console.log(arr); // [3, 4, 5, 7]

Object内部函数

以前ES5时

const person = {
  name: "Taylor",
  sayHello: function() {
    return `Hello! My name is ${this.name}.`;
  }
};

const person = {

  }
};

getter setter

我们可以直接在一个class里面定义一些读取和修改

class people{
    constructor(name,age){
        this.name = name;
        this.age = age;
    }

    get Name(){
        return this.name;
    }

    set Name(newName){
        this.name = newName;
    }
}

要注意属性和方法名要不同，不然会出现递归调用
