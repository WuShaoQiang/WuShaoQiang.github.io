awk的一些用法

常见模式

awk '/reg/' 'BEGIN{} condition {condition action} END{}' filename

其中

  • /reg/ 是一个正则的匹配

  • condition 是判断条件

  • action 就是要执行的操作

  • BEGIN 开始前的操作

  • END 结束后的操作

对文件

awk '{print $1}' test.txt

对管道

df -h | awk '{print $1}'

这里面的$1是指第一部分，默认分隔符是空格(制表符)

echo "This is a test" | awk '{print $1}'

输出的是This

如果想要的分隔符不是默认的，可以使用-F

例如

echo "This,is,a,test" | awk  '{print $1}'

#输出 This,is,a,test

echo "This,is,a,test" | awk -F ',' '{print $1}'

#输出 This

更多高级的用法

先说说内置的变量

NR      行号
NF      有多少列
$1~$n   当前记录的第几个字段

还有一些内置的函数

tolower(str)
toupper(str)
index(str1,str2)
length()
substr()
sqrt()
rand()
...

参考

一个例子

echo "This,is,a,test" \ 
| awk -F ',' 'BEGIN{print "This is Begin"} $0!="" && NF>0 {if (length($0) > 1) print $1} END{print "THIS IS END"}'

可以看出常用模式下的写法
