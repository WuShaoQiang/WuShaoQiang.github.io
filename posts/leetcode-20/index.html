题目

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

    左括号必须用相同类型的右括号闭合。
    左括号必须以正确的顺序闭合。

注意空字符串可被认为是有效字符串。

示例 1:

输入: "()"
输出: true

示例 2:

输入: "()[]{}"
输出: true

示例 3:

输入: "(]"
输出: false

示例 4:

输入: "([)]"
输出: false

示例 5:

输入: "{[]}"
输出: true

我的解答

笔者在做这道题的时候想的稍微复杂了点，用了递归的方法做，但速度什么的非常慢，所以不值得参考，只能说一路摸坑吧

var (
	relation = map[string]string{"(": ")", "[": "]", "{": "}"}
)

func isValid(s string) bool {
	if len(s) == 0 {
		return true
	}
	if (len(s)+2)%2 != 0 {
		return false
	}
	if len(s) == 2 {
		return string(s[1]) == relation[string(s[0])]
	}

	// l := len(s)

	for {
		idx := getLastIndex(s)
		if idx == -1 {
			return false
		}
		if !isValid(s[1:idx]) {
			return false
		}
		if idx == len(s)-1 {
			break
		}
		s = s[idx+1:]
	}
	return true
}

func getLastIndex(s string) int {
	firstStr := string(s[0])
	lastStr, exist := relation[firstStr]
	if !exist {
		return -1
	}
	idx := strings.Index(s, lastStr)
	for {
		count := strings.Count(s[1:idx], firstStr) - strings.Count(s[1:idx], lastStr)
		if count > 0 {
			for count > 0 {
				addIdx := strings.Index(s[idx+1:], lastStr)
				if addIdx == -1 {
					return -1
				}
				idx += (addIdx + 1)
				count--
			}
		} else {
			return idx
		}
	}

}

虽然效果不好，但是总归是脑子的产物，也总结一下吧

核心点在于找到两个对应的括号(这里要主要第一个括号不能对应到第二个括号的反括号，所以找索引这个挺花时间的)，然后把括号里面的再传入，直到出错为止。如果一直没有错，函数就会返回true

递归的思想就是，我每次传入的字符串都是要符合头尾是对应括号的，里面的内容我不管，让递归进去的函数管

最优解答

type Stack struct {
	List *list.List
}

func isValid(s string) bool {
	stack := Stack{List: list.New()}
	for _, v := range s {
		elem := stack.List.Back()
		if elem == nil {
			stack.List.PushBack(string(v))
			continue
		}
		switch elem.Value.(string) {
		case `(`:
			if string(v) != `)` {
				goto Push
			}
			stack.List.Remove(stack.List.Back())
		case `[`:
			if string(v) != `]` {
				goto Push
			}
			stack.List.Remove(stack.List.Back())
		case `{`:
			if string(v) != `}` {
				goto Push
			}
			stack.List.Remove(stack.List.Back())
		}
		continue
	Push:
		stack.List.PushBack(string(v))
	}
	return stack.List.Len() == 0
}

其实这才是这道题的正解，利用栈的思想，如果是有效的括号，那么栈到最后是为空的，因为每个括号都有对应的反括号存在，只要符合对应关系，栈就会把这组数据抛弃掉，最后所有都抛弃就空了。如果不为空就证明对应关系不正确
