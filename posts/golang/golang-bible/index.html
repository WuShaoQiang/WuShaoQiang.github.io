命令行参数

os.Args变量是一个字符串的切片，并且它的第一个元素是命令本身的名字

func main() {
	var s, sep string
	for i := 0; i < len(os.Args); i++ {
		s += sep + os.Args[i]
		sep = " "
	}
	fmt.Println(s)
}

运行go run main.go 123

输出/tmp/go-build115927302/b001/exe/main 123

查找重复的行

  • input := bufio.NewScanner(os.stdin) ~ func NewScanner(r io.Reader) *Scanner

返回一个从r读取的Scanner

  • func (s *Scanner) Scan() bool

逐行扫描，当scan停止的时候，或者是到了读取的终点，又或者是发生了错误，这里都会返回false

  • func (s *Scanner) Text() string

返回scanner所在行的文本

	counts := make(map[string]int)
	input := bufio.NewScanner(os.Stdin)
	for input.Scan() {
		counts[input.Text()]++
	}

暂时发现os.Open()要用绝对路径

  • 读取文件 –Scanner + os.Open() + Scanner.Text() :文本读出，一行一行读 – ioutil.ReadFile(filename): 切片输出，一次读完

JSON

在编码时，默认使用Go语言结构体的成员名字作为JSON对象。只有导出的结构体成员才会被编码，这也就是为什么选择用大写字母开头的成员名称

type Movie struct {
	Title  string
	Year   int  `json:"released"`
	Color  bool `json:"color,omitempty"`
	Actors []string
}

type IssuesSearchResult struct {
	TotalCount int `json:"total_count"`
	Items      []*Issue
}

type Issue struct {
	Number    int
	HTMLURL   string `json:"html_url"`
	Title     string
	State     string
	User      *User
	CreatedAt time.Time `json:"created_at"`
	Body      string    // in Markdown format
}

type User struct {
	Login   string
	HTMLURL string `json:"html_url"`
}

一个结构体成员Tag是和在编译阶段关联到该成员的元信息字符串

解码和编码JSON都可以使用序列化/非序列化方法或者Decoder/Encoder

文本和HTML模板

文本模板(text/template)

const templ = `{{.TotalCount}} issues:
{{range .Items}}----------------------------------------
Number: {{.Number}}
User:   {{.User.Login}}
Title:  {{.Title | printf "%.64s"}}
Age:    {{.CreatedAt | daysAgo}} days
{{end}}`

|操作符表示将前一个表达式的结果作为后一个函数的输入。 上面就是将CreateAt传给了daysAgo作为参数

func daysAgo(t time.Time) int {
	return int(time.Since(t).Hours() / 24)
}

//!-daysAgo

//!+exec
var report = template.Must(template.New("issuelist").
	Funcs(template.FuncMap{"daysAgo": daysAgo}).
	Parse(templ))

func main() {
	result, err := github.SearchIssues(os.Args[1:])
	if err != nil {
		log.Fatal(err)
	}
	if err := report.Execute(os.Stdout, result); err != nil {
		log.Fatal(err)
	}
}
  • template.Must() 简化致命错误的处理
  • template.New()创建并返回一个模板
  • template.Funcs()将自定义函数注册到模板中
  • template.Parse()函数分析模板

获得了模板后，调用func (t *Template) Execute(wr io.Writer, data interface{}) error这个函数 这个函数实现了数据->模板->输出

如果我们信任模板就是HTML，为了防止转义出错(如果把HTML转义成文本就会出错)，可以用template.HTML字符串类型

func main() {
	const templ = `<p>A: {{.A}}</p><p>B: {{.B}}</p>`
	t := template.Must(template.New("escape").Parse(templ))
	var data struct {
		A string        // untrusted plain text
		B template.HTML // trusted HTML
	}
	data.A = "<b>Hello!</b>"
	data.B = "<b>Hello!</b>"
	if err := t.Execute(os.Stdout, data); err != nil {
		log.Fatal(err)
	}
}
